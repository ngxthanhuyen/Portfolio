Site web de météo

Ce site a été réalisé dans le cadre de la SAE 301 (Situation d'Apprentissage et d'Évaluation) par moi-même et deux autres camarades de l'IUT. Les données météorologiques proviennent de l'API SYNOP, une API qui permet d'accéder aux observations météo issues des stations météorologiques professionnelles réparties sur l'ensemble du territoire français, mises à jour régulièrement.

HTML CSS PHP JavaScript MySQL

Fonctionnalités

  • Visualisation données temps réel
  • Recherche médecins par critères
  • Système d'authentification sécurisé

Plateforme web & mobile de soutien scolaire (Edumate)

Développement d'une plateforme permettant de mettre en relation étudiants et tuteurs, de favoriser l'échange de compétences entre pairs et de rendre l'accompagnement plus accessible grâce à l'IA.

Node.js/Express TypeScript MongoDB PostgreSQL Blockchain (Ganache) WebSocket Docker JWT

Fonctionnalités clés

  • Recherche avancée : Filtrage par matière, niveau, tarif et localisation
  • Gestion des annonces : Publication et modération des offres de cours
  • Messagerie temps réel : Communication directe étudiants-tuteurs
  • Échange de compétences : Entraide sans transaction financière
  • Recherche sémantique IA : Service RAG avec Qdrant pour recommander les tuteurs pertinents
  • E-wallet sécurisé : Gestion des crédits virtuels avec traçabilité via blockchain
  • Analyse automatique des CV : Extraction intelligente des compétences
  • Assistant IA : Disponible 24/7 pour répondre aux questions des utilisateurs et offrir une assistance personnalisée
